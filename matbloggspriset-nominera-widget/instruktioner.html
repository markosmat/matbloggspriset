Matbloggspriset — Nordens äldsta bloggpris

Installation i HTML-dokument, Blogger, Blogg.se, WordPress.com och andra bloggtjänster

  1. Lägg Nominera mig-boxen på din sida. I detta HTML-dokument kan du se ett exempel uppe vid det högra hörnet.
  2. Inkludera koden någonstans i din HTML- eller mallfil efter </head> och innan </body>
    3. Kodsnutt att klistra in<a target='_blank' class='matbloggspriset-nominera-mig' href='http://www.matbloggspriset.se/nominera' title='Nominera mig till Matbloggspriset Folkets val!'><img src='https://raw.github.com/Deluxive/matbloggspriset/master/matbloggspriset-nominera-widget/nominera_mig_matbloggspriset.png' alt='Nominera mig till Matbloggspriset Folkets val' style='position: fixed; top: 28px; right: 0; z-index: 100000; cursor: pointer; border:none;' /></a>
  3. Detta är det enklaste sättet, då det inkluderar både bildfilen, HTML och CSS. I vissa fall kan det dock vara bättre att separera dessa genom att inkludera de var för sig:
    4. matbloggspriset_nominera_mig.css mellan <head> </head> <link href='https://raw.github.com/Deluxive/matbloggspriset/master/matbloggspriset-nominera-widget/matbloggspriset_nominera_mig.css' rel='stylesheet' type='text/css'>
  4. Därefter så klistrar du in följande någonstans i ditt HTML-dokument eller mall:
    5. <a id='nomineramig' href='http://www.matbloggspriset.se/nominera' target='_blank'> <img alt='Nominera mig till Matbloggspriset - Folkets val!' src='https://raw.github.com/Deluxive/matbloggspriset/master/matbloggspriset-nominera-widget/nominera_mig_matbloggspriset.png' width='178' height='178'>
  5. Spara och uppdatera din webbsida

Förklaring av innehållet i CSS-filen

    CSS-filen är ytterst enkel och består av en enda rad #nomineramig { position: fixed; top: 28px; right: 0; z-index: 10000; cursor: pointer; border:none; }
  1. #nomineramig är själva behållaren för ”Nominera mig!”-grafiken som syns i webbläsaren.
  2. position: fixed; Gör att ”Nominera mig!”-grafiken är kvar på samma ställe även om du scrollar.
  3. top: 28px; right: 0; Är placeringen på webbsidan. 28 pixlar från toppen och 0 pixlar från högersida av fönstret.
  4. z-index: 10000; Gör så att ”Nominera mig!”-grafiken alltid är det översta ”lagret” på sidan. Om du sätter z-index till t.ex. 3 och annat innehåll på sidan har z-index: 4;, så kommer det innehållet med z-index: 4; lägga sig över.
  5. cursor: pointer; Visar pekande hand istället för den vanliga muspekaren.
  6. border:none; Vissa webbläsare lägger till en ram på bilder. Detta tar bort den.
